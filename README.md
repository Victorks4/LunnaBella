## 📋 Descrição

**Luna Bella** é uma plataforma de e-commerce especializada em moda feminina, oferecendo uma experiência elegante e moderna para mulheres que buscam peças atemporais, estilo jovem e moda modesta. O projeto combina design sofisticado com funcionalidades práticas de compra online.

## ✨ Funcionalidades

### 🛍️ Catálogo de Produtos
- **Categorias Especializadas**: Atemporais, Jovem e Modesta
- **Sistema de Busca**: Pesquisa inteligente por produtos
- **Filtros Avançados**: Por categoria, preço, tamanho e cor
- **Visualização Detalhada**: Imagens em alta qualidade com zoom

### 🛒 Carrinho de Compras
- **Carrinho Lateral**: Interface intuitiva e responsiva

## 🚀 Instalação e Configuração

### Pré-requisitos
- Node.js (versão 16 ou superior)
- NPM ou Yarn

### Passos para Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/seu-usuario/LunnaBella.git
   cd LunnaBella
   ```

2. **Instale as dependências**
   ```bash
   npm install
   # ou
   yarn install
   ```

3. **Execute o projeto**
   ```bash
   # Para desenvolvimento
   npm run dev
   # ou
   yarn dev
   ```

4. **Acesse o projeto**
   - Abra `index.html` no navegador
   - Ou use um servidor local como Live Server
